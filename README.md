# pong-2d-godot
Remake simples do clássico Pong feito com a engine Godot como parte do meu aprendizado em desenvolvimento de jogos.

Controles 
W/S para subir/descer o jogador 1
↑/↓ para subir/descer o jogador 2
Esc para sair
R para Reiniciar

Tecnologias
- Godot Engine 4.4.1
- GDScript
